Creating a New Documents –

Now, after learning about ribbons bar, ruler bar and text area it’s a time to create a new documents. To create a new documents you first need know about what kind of documents you are going to create.

These are some steps to create a documents –

  1. Click on File at Menu bar.
  2. Select the New option.
  3. Here you get many template choose any one based on your document format. (images of types of formats)
  4. Select the desired one and click OK.
