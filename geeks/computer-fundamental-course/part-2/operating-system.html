What is Operating System ?

An Operating System is the important part of computer system, which controls the all others component associated with system.

Some Major component is -

  • The Hardware
  • The Operating System
  • The Application Program Routine (Compiler, Interpreter etc.)
  • The Human ware (Users)

Or in other world you can says that -
An Operating System is a program, which acts as interface between users and the hardware of computer system.

When Hardware provides the all basic computing resources, the Application program routines define the ways for using these resources and to solve the computing problems of Users. And here, Operating System is used to control and co-ordinate the use of hardware among various program for single or multiple users.

Functions of Operating System

The Operating System provide several services to program and the user of that program, some common service which are provided by Operating System are listed below -

  • Program Execution
  • Program Execution
  • Input-Output Operation Handling
  • Manipulation of File System
  • Error Handling
  • Resources Allocation
  • Accounting
  • Information And Resources Protection

Now ready to learn about each one -

Program Execution -
Operating System is responsible for executing various program, may be system programs or user programs. Execution is Operating system responsibility.

Input-Output Operation Handling -
The Operating System handle the various types of Inputs and various types of Output which was gives by users and after processing shown by system respectively.

Manipulation of File System -
Decision you take regarding the storage of files and folder are involves in this process.

Error Handling -
Operating System is also helps to detect any kind of error and to handle or resolve them.

Resources Allocation -
This involves the proper uses of available resources.

For Example -

If multiple files selected for print by the users then, how and in which order they are print ? Is decided by Operating System.

Accounting - Operating System keeps an account of what types of function are performed by system and what kind of error have occurred.

Information and Resources Protection - The Operating System is responsible for ensuring that all the information and available resources are used in a correct way.
